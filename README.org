#+title: khcalc

This is an ongoing project building toward an implementation of Queffelec-Rose style constructions of colored sl_{n}/gl_{n} link invariants.

** Current Status
khcalc currently provides the HOMFLY polynomial of a braid closure from the Hecke algebra directly. Below is an example calculation for the right-handed trefoil.

#+begin_src python :results output
from braid import BraidElem
from trace import homfly_from_braid as homfly

tref = BraidElem(2, (1,1,1)) #right-handed trefoil
print(homfly(tref))

#OUTPUT: (-q^-3*a^-2 + q^-1*a^-4 + q^-1*a^-2 - q*a^-4 - q*a^-2 + q^3*a^-2)/(-q^-1 + q)
#+end_src
